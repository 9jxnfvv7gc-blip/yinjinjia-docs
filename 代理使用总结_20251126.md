## 🎯 关于访问 Google

### 方案1：临时开启代理（推荐）

**仅在需要访问 Google 时开启代理，使用完后立即关闭。**

#### 使用步骤：

1. **需要访问 Google 时**：
   - iPhone 设置 → Wi-Fi → 点击 ⓘ → HTTP 代理 → 手动
   - 服务器：`47.243.177.166`
   - 端口：`3128`
   - 保存

2. **访问 Google**：
   - 打开 Safari
   - 访问 `https://www.google.com`
   - 完成需要的操作（注册、验证等）

3. **立即关闭代理**：
   - iPhone 设置 → Wi-Fi → 点击 ⓘ → HTTP 代理 → 关闭
   - 恢复正常网络使用

### 方案2：使用远程桌面（更稳定）

**所有 Google 相关操作在远程桌面中完成，无需在手机上配置代理。**

#### 优势：
- ✅ 更稳定可靠
- ✅ 不需要频繁开关代理
- ✅ 不影响手机正常使用
- ✅ 所有操作在服务器端完成

#### 使用步骤：

1. **连接远程桌面**（Mac 上的 Windows App）
   - PC 名称：`47.243.177.166`
   - 用户名：`desktop`
   - 密码：你的服务器密码

2. **在远程桌面中**：
   - 打开 Firefox 浏览器
   - 访问 `https://accounts.google.com/signup`
   - 完成注册流程

3. **关于手机验证**：
   - 如果提示手机验证，使用备用邮箱验证
   - 或使用语音验证（电话接收验证码）
   - 验证码是数字，在远程桌面浏览器中输入
