# 📱 真实Android设备测试指南

## 🔌 连接真实设备

### 步骤1：启用USB调试

1. **打开手机设置**

2. **关于手机**：
   - 设置 → 关于手机
   - 找到"版本号"或"内部版本号"

3. **启用开发者模式**：
   - 连续点击"版本号"**7次**
   - 会提示"您已成为开发者"

4. **返回设置**：
   - 返回设置主页面
   - 找到"开发者选项"（通常在"系统"或"关于手机"附近）

5. **启用USB调试**：
   - 打开"开发者选项"
   - 启用"USB调试"
   - 可能还需要启用"USB安装"（如果提示）

---

### 步骤2：连接设备

1. **用USB线连接手机和电脑**

2. **在手机上确认**：
   - 会弹出"允许USB调试"对话框
   - 勾选"始终允许来自这台计算机"
   - 点击"确定"

---

### 步骤3：检查连接

在终端运行：

```bash
adb devices
```

**应该看到**：
```
List of devices attached
<设备ID>    device
```

如果显示"unauthorized"：
- 在手机上重新确认"允许USB调试"

---

### 步骤4：运行应用

```bash
cd /Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905
flutter run -d android
```

**或者指定设备ID**：
```bash
flutter run -d <设备ID>
```

## ⚠️ 常见问题

### 问题1：设备未识别

**解决方法**：
1. 检查USB线是否正常
2. 尝试不同的USB端口
3. 在手机上重新确认"允许USB调试"
4. 重启adb：
   ```bash
   adb kill-server
   adb start-server
   adb devices
   ```

---

### 问题2：权限问题

**解决方法**：
1. 确保已启用"USB调试"
2. 确保已启用"USB安装"（如果需要）
3. 在手机上重新确认权限

---

### 问题3：应用安装失败

**解决方法**：
1. 检查设备存储空间
2. 检查设备Android版本（需要Android 5.0+）
3. 检查应用权限设置
