# 购买香港服务器指南

## 如何购买香港服务器

### 主要服务商

#### 1. 阿里云（Alibaba Cloud）
- **官网**：https://www.aliyun.com
- **优势**：
  - 中文界面，操作简单
  - 支持支付宝/微信支付
  - 有详细的中文文档
  - 价格相对透明
- **购买步骤**：
  1. 注册账号
  2. 实名认证
  3. 选择"云服务器ECS"
  4. 选择地域：香港
  5. 选择配置（最低1核1G即可）
  6. 选择操作系统（推荐Ubuntu）
  7. 购买

#### 2. 腾讯云（Tencent Cloud）
- **官网**：https://cloud.tencent.com
- **优势**：
  - 中文界面
  - 支持微信支付
  - 有新手优惠
- **购买步骤**：类似阿里云

#### 3. 其他服务商
- **AWS（Amazon Web Services）**：国际知名，但配置较复杂
- **DigitalOcean**：简单易用，但需要信用卡
- **Vultr**：价格便宜，支持按小时计费

### 推荐配置

#### 最低配置（适合测试）
- **CPU**：1核
- **内存**：1GB
- **硬盘**：20GB
- **带宽**：1-5Mbps
- **价格**：约 50-100 元/月

#### 推荐配置（适合长期使用）
- **CPU**：2核
- **内存**：2GB
- **硬盘**：40GB
- **带宽**：5-10Mbps
- **价格**：约 100-200 元/月

### 购买注意事项

1. **选择按量付费还是包月**
   - 按量付费：适合偶尔使用
   - 包月：适合长期使用，更经济

2. **选择操作系统**
   - 推荐：Ubuntu 20.04 或 22.04
   - 原因：易于配置，文档丰富

3. **选择地域**
   - 必须选择：香港
   - 其他地域可能无法访问 Google 服务

4. **安全组设置**
   - 开放必要的端口（如SSH的22端口）
   - 配置防火墙规则

## 购买后的步骤

### 1. 连接到服务器
```bash
# 使用SSH连接
ssh root@你的服务器IP
```

### 2. 安装代理软件
可以选择：
- **Shadowsocks**：简单易用
- **V2Ray**：功能强大
- **Trojan**：安全性高

### 3. 配置代理
按照代理软件的文档进行配置

### 4. 在本地连接
- 下载代理客户端
- 配置服务器信息
- 连接到服务器

## 成本估算

### 按量付费
- **CPU/内存**：约 0.1-0.3 元/小时
- **流量**：约 0.8 元/GB
- **适合**：偶尔使用

### 包月
- **最低配置**：约 50-100 元/月
- **推荐配置**：约 100-200 元/月
- **适合**：长期使用

## 新手建议

1. **先试用按量付费**
   - 测试是否满足需求
   - 避免浪费

2. **选择中文服务商**
   - 阿里云或腾讯云
   - 有中文客服支持

3. **参考教程**
   - 服务商通常有详细教程
   - 可以搜索"香港服务器搭建代理"教程
