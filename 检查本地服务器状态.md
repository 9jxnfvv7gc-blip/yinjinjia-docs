# 检查本地服务器状态

## 🔍 如何检查服务器是否运行

### 方法1：检查终端窗口

**查看显示 `python3 video_server.py` 的终端窗口**：

- **如果看到**：
  ```
  Serving at port 8081
  启动视频服务器...
  端口: 8081
  视频根目录: /Volumes/Expansion
  ```
  **说明**：✅ 服务器正在运行

- **如果看到**：
  ```
  Traceback (most recent call last):
  ...
  ```
  **说明**：❌ 服务器启动失败

- **如果没有任何输出**：
  **说明**：❌ 服务器可能没有启动

---

### 方法2：测试服务器连接

在**新的终端窗口**执行：

```bash
# 测试服务器是否响应
curl http://localhost:8081
```

**如果看到HTML内容**：
- ✅ 服务器正在运行

**如果看到错误**：
- ❌ 服务器没有运行

---

### 方法3：检查进程

在**新的终端窗口**执行：

```bash
# 检查是否有Python服务器进程
ps aux | grep video_server.py | grep -v grep
```

**如果看到进程**：
- ✅ 服务器正在运行

**如果没有输出**：
- ❌ 服务器没有运行

---

## ✅ 如果服务器没有运行

### 启动服务器

在终端执行：

```bash
cd "/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905"
python3 video_server.py
```

**应该看到**：
```
启动视频服务器...
端口: 8081
视频根目录: /Volumes/Expansion

访问 http://localhost:8081 查看服务器状态
在 Flutter App 中输入服务器地址: http://你的IP:8081

按 Ctrl+C 停止服务器

Serving at port 8081
```

**保持这个终端窗口打开**，不要关闭。

---

## 🔄 如果服务器已经在运行

### 确认状态

1. **查看终端输出**：应该看到 "Serving at port 8081"
2. **测试连接**：在浏览器访问 `http://localhost:8081`
3. **在应用中刷新**：下拉刷新应用，应该能看到视频和歌曲

---

## 📋 快速检查步骤

### 步骤1：检查终端

查看显示 `python3 video_server.py` 的终端窗口：

1. **服务器需要一直运行**：关闭终端窗口会停止服务器
2. **保持终端打开**：不要关闭运行服务器的终端窗口
3. **如果关闭了**：重新运行启动命令
