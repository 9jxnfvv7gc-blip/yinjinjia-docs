## 🔧 配置位置

文件：`lib/config.dart`

```dart
/// 是否使用示例内容（不连接服务器，直接显示示例视频和音乐）
static const bool useSampleContent = true;
```

## 🔄 切换模式

### 使用示例内容（当前）
```dart
static const bool useSampleContent = true;
```

### 连接服务器获取真实内容
```dart
static const bool useSampleContent = false;
```
