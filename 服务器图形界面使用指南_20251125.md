## 🔌 连接远程桌面

### 步骤1：在 Mac 上打开 Windows App

1. 打开 **Windows App**（原 Microsoft Remote Desktop）
2. 添加或选择连接：
   - **PC name**：`47.243.177.166:3389`
   - **User account**：`desktop`（你创建的用户）
   - **Password**：你设置的密码

### 步骤2：登录远程桌面

1. 双击连接
2. 在登录界面：
   - **Session**：选择 `Xorg`
   - **Username**：`desktop`
   - **Password**：输入密码
3. 点击 OK 登录

### 步骤3：进入 XFCE 桌面

登录成功后，你会看到 XFCE 桌面环境。

## 🌐 使用 Firefox 浏览器

### 方法1：通过应用程序菜单

1. 在 XFCE 桌面左上角，点击 **Applications**（应用程序）
2. 选择 **Internet** → **Firefox Web Browser**

### 方法2：通过终端启动

1. 在桌面空白处右键 → **Open Terminal Here**
2. 输入命令：
   ```bash
   firefox &
   ```
3. Firefox 会在后台启动

### 方法3：直接访问 Google Play Console

在终端中直接打开：
```bash
firefox https://play.google.com/console &
```

## 🔧 如果 Firefox 无法启动

### 检查1：确认 Firefox 已安装
```bash
which firefox
firefox --version
```

### 检查2：尝试直接运行
```bash
/usr/bin/firefox
```

### 检查3：如果是 snap 版本
```bash
snap list | grep firefox
/snap/bin/firefox
```

### 检查4：安装其他浏览器（备选）
如果 Firefox 有问题，可以安装 Chromium：
```bash
sudo apt update
sudo apt install -y chromium-browser
```

然后在应用程序菜单：**Internet** → **Chromium Web Browser**

## 🐛 常见问题

### 问题1：Firefox 启动很慢
- **原因**：snap 版本首次启动需要初始化
- **解决**：等待 30-60 秒，或使用 Chromium

### 问题2：无法访问 Google
- **检查**：服务器网络连接
- **测试**：在终端运行 `ping google.com`
- **解决**：确保服务器可以访问外网

### 问题3：远程桌面连接失败
- **检查**：UFW 防火墙状态 `sudo ufw status`
- **检查**：xrdp 服务状态 `sudo systemctl status xrdp`
- **检查**：阿里云安全组是否开放 3389 端口

### 问题4：Firefox 显示黑屏或无法显示
- **解决**：尝试重启远程桌面会话
- **或**：使用 Chromium 浏览器
