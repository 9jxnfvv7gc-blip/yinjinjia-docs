## 🎯 下一步操作

### 步骤1：选择 Proxy 代理组

1. **在 Proxies 标签中**
2. **点击选择 "Proxy" 代理组**
3. **确认已选择**（应该会高亮显示）

### 步骤2：找到并开启系统代理

系统代理开关可能在以下位置：

#### 位置1：主窗口顶部

1. **查看 Clash Verge 主窗口的顶部**
2. **查找"系统代理"、"System Proxy"或类似按钮**
3. **点击开启**（按钮应该会变成开启状态）

#### 位置2：菜单栏图标

1. **点击菜单栏的 Clash Verge 图标**（右上角）
2. **查找"系统代理"、"System Proxy"或类似选项**
3. **点击开启**

#### 位置3：设置页面

1. **点击"设置"或"Settings"标签**
2. **查找"系统代理"、"System Proxy"或类似选项**
3. **开启它**

### 步骤3：测试 Mac 访问 Google

1. **在 Mac 上打开浏览器**（Safari 或 Chrome）
2. **访问**：`https://www.google.com`
3. **确认可以访问**（如果能看到 Google 首页，说明配置成功 ✅）

---

## 🔍 如果找不到系统代理开关

### 查看 Clash Verge 主窗口

1. **查看 Clash Verge 主窗口**
2. **应该有多个标签**：Proxies, Profiles, Settings, Connections 等
3. **切换到 Settings 标签**
4. **查找系统代理选项**

### 或者查看菜单栏

1. **查看菜单栏是否有 Clash Verge 图标**
2. **点击图标查看菜单**
3. **查找系统代理选项**

---

## 📝 如果 Mac 可以访问 Google

### 下一步：开启局域网共享

1. **打开 Clash Verge 设置**
2. **找到"允许局域网连接"或"Allow LAN"**
3. **✅ 勾选这个选项**
4. **记录 HTTP 端口**：7890

### 然后配置 iPhone

1. **查看 Mac 内网 IP**：
   ```bash
   ipconfig getifaddr en0
   ```

2. **在 iPhone 上配置 Wi-Fi 代理**：
   - 设置 → Wi-Fi → 点击当前网络 ⓘ
   - HTTP 代理 → 手动
   - 服务器：Mac 的内网 IP
   - 端口：7890
   - 保存

3. **测试 iPhone 访问 Google**
