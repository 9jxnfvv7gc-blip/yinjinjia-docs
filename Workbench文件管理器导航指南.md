# Workbench文件管理器导航指南

## 📁 导航到 /root/ 目录的方法

### 方法1：使用地址栏（最简单）

1. 在文件管理器窗口的**顶部**，找到**地址栏**（显示当前路径的地方）
2. 点击地址栏，清空当前路径
3. 输入：`/root`
4. 按**回车键**（Enter）

**应该会显示**：`/root/` 目录的内容

---

### 方法2：使用目录树（左侧导航）

1. 在文件管理器的**左侧**，找到**目录树**（文件夹列表）
2. 如果看到 `root` 文件夹，直接点击它
3. 如果看不到 `root`，可能需要：
   - 点击 **根目录** 或 **/** 图标
   - 或者展开 `/` 目录，找到 `root` 文件夹
   - 点击 `root` 文件夹

---

### 方法3：使用"向上"按钮

1. 在文件管理器顶部工具栏，找到**向上**按钮（通常是向上箭头图标 `↑` 或 `../`）
2. 多次点击**向上**按钮，直到到达 `/root/` 目录
3. 或者直接点击**根目录**按钮（`/` 图标）

---

### 方法4：使用终端命令（如果文件管理器不支持）

如果文件管理器无法导航，可以在Workbench终端执行：

```bash
# 创建目录（如果还没有）
mkdir -p /root/video_server
mkdir -p /root/videos/原创视频
mkdir -p /root/videos/原创歌曲

# 验证目录已创建
ls -la /root/
```

然后刷新文件管理器页面，应该能看到这些目录。

---

## 🎯 推荐操作流程

### 步骤1：导航到 /root/

**使用地址栏**：
1. 在文件管理器顶部找到地址栏
2. 输入：`/root`
3. 按回车键

---

### 步骤2：创建目录（如果还没有）

**方法A：在文件管理器中创建**
1. 在 `/root/` 目录中，右键点击空白处
2. 选择 **新建文件夹** 或 **New Folder**
3. 输入文件夹名称：`video_server`
4. 按回车键确认

**方法B：使用终端创建（推荐）**
在Workbench终端执行：

```bash
# 创建所有需要的目录
mkdir -p /root/video_server
mkdir -p /root/videos/原创视频
mkdir -p /root/videos/原创歌曲

# 验证
ls -la /root/
```

---

### 步骤3：上传文件

1. 导航到 `/root/video_server/` 目录（在地址栏输入 `/root/video_server`）
2. 点击**上传**按钮（通常在顶部工具栏）
3. 选择 `video_server.py` 文件
4. 等待上传完成

---

## 💡 如果找不到地址栏

### 检查文件管理器界面：

1. **顶部工具栏**：查看是否有显示路径的文本框
2. **面包屑导航**：查看是否有类似 `首页 > root` 的导航路径，可以点击
3. **搜索框**：有些文件管理器有搜索功能，可以搜索 `root`

---

## 🔍 如果还是找不到

### 使用终端创建目录和上传：

在Workbench终端执行：

```bash
# 1. 创建目录
mkdir -p /root/video_server
mkdir -p /root/videos/原创视频
mkdir -p /root/videos/原创歌曲

# 2. 验证目录
ls -la /root/

# 3. 检查video_server.py是否已存在
ls -la /root/video_server/
```

**如果 `video_server.py` 已经存在**，说明之前已经上传过了，可以直接使用。

---
