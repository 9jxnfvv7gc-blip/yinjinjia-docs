# 🚀 ClashX 快速安装指南（2025-11-26）

## 📥 下载 ClashX

我已经在浏览器中打开了 ClashX 的下载页面。

**请按以下步骤操作**：

### 步骤1：下载 ClashX

1. **在浏览器中找到 ClashX 的下载页面**（应该已经自动打开了）

2. **找到最新的版本**（通常在页面最上面，例如：`v1.95.1` 或更高）

3. **展开 "Assets" 部分**（点击 "Assets" 展开）

4. **找到并点击下载 `ClashX.dmg` 文件**

5. **等待下载完成**（文件大小约 20-30MB）

---

### 步骤2：安装 ClashX

1. **打开下载的 `ClashX.dmg` 文件**（在 Downloads 文件夹中）

2. **将 ClashX 拖拽到 Applications 文件夹**

3. **打开 Applications 文件夹，找到 ClashX**

4. **右键点击 ClashX → 打开**（首次打开需要确认，点击"打开"）

5. **在菜单栏看到 ClashX 图标**（小猫咪 🐱 图标）

---
