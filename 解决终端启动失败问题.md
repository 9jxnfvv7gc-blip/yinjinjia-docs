# 解决终端启动失败问题

## ❌ 问题描述

**错误信息**：
```
终端进程启动失败: 启动目录(cwd)"/Users/xiaohuihu/Desktop/桌面影音播放器_安装包_20251121_165905/macos"不存在。
```

**原因**：
- Cursor IDE 的工作目录配置错误
- 项目实际不在 `/Users/xiaohuihu/Desktop/...` 路径下
- 正确的项目路径应该是：`/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905`

---

## ✅ 解决方案

### 方案1：在 Cursor 中重新打开正确的项目文件夹（推荐）

#### 步骤：

1. **关闭当前工作区**
   - 在 Cursor 中：`File → Close Folder` 或 `File → Close Workspace`

2. **打开正确的项目文件夹**
   - 在 Cursor 中：`File → Open Folder...`
   - 导航到：`/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905`
   - 点击"打开"

3. **验证工作目录**
   - 打开终端（Cursor 的集成终端）
   - 执行：`pwd`
   - 应该显示：`/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905`

---

### 方案2：使用系统终端（不依赖 IDE）

1. **打开 macOS 系统终端**（Terminal.app）

2. **启动服务器**（第一个终端窗口）：
   ```bash

   ```

3. **运行应用**（第二个终端窗口）：

   flutter run -d macos
