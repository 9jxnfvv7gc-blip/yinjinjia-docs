# 退出日志查看器并创建目录

## 🔍 当前状态

你正在查看 `journalctl` 的日志输出，这是一个分页查看器（类似 `less`）。

---

## 🚀 操作步骤

### 步骤1：退出日志查看器

在终端中：

1. **按 `q` 键**（退出日志查看器）
2. 应该会返回到正常的命令提示符：`root@iZj6cg78ov73x6cxbephc1Z:~/video_server#`

---

### 步骤2：创建目录

退出日志查看器后，在终端执行：

```bash
# 创建所有需要的目录
mkdir -p /root/video_server
mkdir -p /root/videos/原创视频
mkdir -p /root/videos/原创歌曲

# 验证目录已创建
ls -la /root/
```

---

### 步骤3：检查目录结构

```bash
# 查看 /root/ 目录
ls -la /root/

# 查看 video_server 目录
ls -la /root/video_server/

# 查看 videos 目录
ls -la /root/videos/
```

---

## 📋 完整操作流程

### 在Workbench终端执行：

1. **按 `q` 键退出日志查看器**
2. **等待命令提示符出现**：`root@iZj6cg78ov73x6cxbephc1Z:~/video_server#`
3. **执行创建目录命令**：

```bash
mkdir -p /root/video_server
mkdir -p /root/videos/原创视频
mkdir -p /root/videos/原创歌曲
ls -la /root/
```

---

## 💡 关于文件管理器

如果Workbench文件管理器没有地址栏和目录树，可能是：

1. **版本不同**：不同版本的Workbench界面可能不同
2. **可以使用终端**：所有操作都可以通过终端完成
3. **上传文件**：可以使用 `scp` 或 `rsync` 从Mac上传文件
