# 🗑️ 如何删除上传的内容

## ✅ 删除功能已经存在！

删除功能已经实现了，但需要通过**长按**来触发。

---

## 📱 在手机端（iOS/Android）删除内容

### 方法：长按内容卡片

1. **找到要删除的视频或音乐**
   - 在应用列表中
   - 找到你想删除的内容

2. **长按内容卡片**
   - 用手指**长按**（按住不放）内容卡片
   - 等待1-2秒

3. **选择删除**
   - 会弹出底部菜单
   - 显示：
     - 📤 **分享链接**
     - 🔗 **复制链接**
     - 🗑️ **删除**（红色，仅管理员可见）

4. **确认删除**
   - 点击"删除"
   - 会弹出确认对话框
   - 点击"删除"确认

5. **完成**
   - 内容会被删除
   - 显示"删除成功！"提示
   - 列表自动刷新

---

## 💻 在电脑端（macOS）删除内容

### 方法：右键点击或长按

1. **找到要删除的内容**
   - 在应用列表中

2. **右键点击内容卡片**
   - 或者用鼠标**长按**（按住不放）
   - 等待1-2秒

3. **选择删除**
   - 会弹出底部菜单
   - 点击"删除"

4. **确认删除**
   - 点击"删除"确认

---

## ⚠️ 重要提示

### 删除功能需要管理员权限

- 只有管理员才能看到删除按钮
- 当前配置：`isAdmin = true`（开发阶段）
- 生产环境应该从登录状态获取

### 删除操作不可恢复

- 删除后无法恢复
- 请谨慎操作

## 🔍 如果看不到删除按钮

### 检查1：确认是管理员

打开 `lib/config.dart`，确保：
```dart
static bool get isAdmin {
  return true; // 确保是 true
}
```

### 检查2：确认管理员模式已启用

```dart
static const bool enableAdminMode = true;
```

### 检查3：重新加载应用

在终端按 `r`（热重载）或 `R`（完全重启）
