# 🔧 豆包网站无法访问排查（2025-11-26）

## ❌ 问题

关闭系统代理后，还是打不开豆包。

---

## 🔍 排查步骤

### 步骤1：检查网络连接

在终端运行：

```bash
ping -c 3 www.doubao.com
```

如果无法 ping 通，可能是网络或 DNS 问题。

### 步骤2：检查 DNS

在终端运行：

```bash
nslookup www.doubao.com
```

如果无法解析，可能是 DNS 问题。

### 步骤3：测试其他网站

访问其他网站（如 baidu.com），看是否能打开。

如果其他网站也打不开，可能是网络问题。

---

## ✅ 解决方案

### 方法1：清除浏览器缓存

1. **Safari → 设置 → 隐私**
2. **清除历史记录和网站数据**
3. **重新访问豆包网站**

### 方法2：重新连接 Wi-Fi

1. **关闭 Wi-Fi**
2. **等待 10 秒**
3. **重新打开并连接 Wi-Fi**
4. **重新访问豆包网站**

### 方法3：检查 DNS 设置

1. **系统设置 → 网络**
2. **选择当前网络连接**
3. **点击"高级"**
4. **切换到"DNS"标签**
5. **检查 DNS 服务器设置**
6. **可以添加**：8.8.8.8 或 114.114.114.114

### 方法4：重启网络服务

在终端运行：

```bash
sudo dscacheutil -flushcache
sudo killall -HUP mDNSResponder
```

---

## 🔍 如果其他网站也打不开

可能是网络连接问题：

1. **检查网络连接**
2. **重新连接 Wi-Fi**
3. **或检查路由器设置**
