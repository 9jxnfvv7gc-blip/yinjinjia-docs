# 🔍 香港服务器视频检查

## ✅ 服务器状态

- ✅ 服务器已启动（进程运行中）
- ✅ API 可以访问
- ❌ 返回空数组 `[]`，说明没有找到视频文件

## 🔍 在服务器终端执行以下命令

### 检查视频文件位置

```bash
# 检查视频目录
ls -la /root/videos/原创视频/

# 查找所有视频文件
find /root/videos -type f \( -name "*.mp4" -o -name "*.mov" -o -name "*.mkv" \) 2>/dev/null | head -10

# 检查目录结构
ls -laR /root/videos/ | head -30

# 检查其他可能的视频位置
find /root -type d -name "*video*" -o -name "*视频*" 2>/dev/null
```

## 💡 可能的情况

1. **视频文件在其他目录**
   - 需要找到实际位置
   - 然后修改 `VIDEO_ROOT` 环境变量

2. **视频文件还没有上传**
   - 需要上传视频文件到 `/root/videos/原创视频/` 目录

3. **视频文件在其他服务器**
   - 需要从其他位置复制过来

## 🔧 如果找到视频文件在其他位置

例如视频在 `/root/my_videos/`：

```bash
# 停止旧服务器
pkill -f video_server.py

# 修改 VIDEO_ROOT
export VIDEO_ROOT="/root/my_videos"

# 重新启动服务器
cd /root/app
nohup python3 video_server.py > /root/server.log 2>&1 &

# 测试
curl http://localhost:8081/api/list/原创视频
```

## 📤 如果需要上传视频文件

如果视频在本地 Mac，可以上传：

```bash
# 在本地 Mac 终端执行
scp /本地视频路径/*.mp4 root@47.243.177.166:/root/videos/原创视频/
```
