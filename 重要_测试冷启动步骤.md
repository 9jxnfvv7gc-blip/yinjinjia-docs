## 📱 如果应用闪退，使用 Xcode 查看崩溃日志

### 方法 1：使用 Xcode Console（推荐）

1. **打开 Xcode**（我已经帮你打开了项目）
2. **连接 iPhone**：用 USB 连接
3. **运行应用**：
   - 点击运行按钮（▶️）或按 `Cmd + R`
   - 应用启动后，点击停止按钮（⏹️）或按 `Cmd + .`
4. **打开 Console**：按 `Shift + Cmd + C`
5. **重启手机** → 解锁
6. **直接点击应用图标**
7. **观察 Console 输出**：如果崩溃，会显示崩溃信息

### 方法 2：查看设备日志

1. **打开 Xcode**：`Window → Devices and Simulators`（或按 `Shift + Cmd + 2`）
2. **选择设备**：在左侧选择你的 iPhone
3. **查看日志**：点击 "View Device Logs"
4. **找到最新日志**：按时间排序，最新的在最上面
5. **双击打开**：查看完整的崩溃堆栈
