# 🔍 查找视频文件 - 恢复分享、删除功能

## 📋 目标

找到原来的视频文件，恢复应用的分享、删除等功能。

## 🔍 在服务器终端执行以下命令

### 查找所有视频文件

```bash
# 在服务器上查找所有视频文件
find /root -type f \( -name "*.mp4" -o -name "*.mov" -o -name "*.mkv" -o -name "*.avi" \) 2>/dev/null

# 查找所有目录
find /root -type d 2>/dev/null | grep -E "video|视频|content|media" | head -20

# 检查 /root 下所有目录
ls -la /root/ | grep "^d"

# 检查是否有其他用户目录
ls -la /home/
```

### 检查可能的视频位置

```bash
# 检查常见位置
ls -la /var/www/
ls -la /opt/
ls -la /usr/local/
ls -la /tmp/

# 检查是否有挂载的磁盘
df -h
mount | grep -v "^/dev"
```

## 💡 可能的位置

1. **在其他目录**：`/root/my_videos/`, `/root/content/`, `/root/media/`
2. **在其他用户目录**：`/home/admin/videos/`
3. **在其他磁盘**：挂载的外部存储
4. **在其他服务器**：可能还在香港服务器的其他位置

## 🎯 找到后

如果找到视频文件，例如在 `/root/my_videos/`：

```bash
# 修改 VIDEO_ROOT
export VIDEO_ROOT="/root/my_videos"

# 停止旧服务器
pkill -f video_server.py

# 重新启动
cd /root/app
nohup python3 video_server.py > /root/server.log 2>&1 &

# 测试
curl http://localhost:8081/api/list/原创视频
```
