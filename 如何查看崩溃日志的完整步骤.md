# 📱 如何查看崩溃日志的完整步骤

## 🎯 方法 1：使用 flutter run（推荐）

### 步骤 1：启动应用并查看日志

在终端执行：
```bash
cd "/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905"
flutter run -d Dianhua
```

这会：
- 构建并安装应用
- 启动应用
- **实时显示所有日志**（包括崩溃信息）

### 步骤 2：测试冷启动

1. **应用启动后，按 `q` 退出**（应用会留在手机上）
2. **重启手机** → 解锁
3. **直接点击桌面上的应用图标**
4. **观察终端输出**：
   - 如果崩溃，终端会显示崩溃堆栈
   - 如果正常启动，终端会显示应用日志

## 🎯 方法 3：使用 Xcode（最详细）

### 步骤 1：打开 Xcode

```bash
cd "/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905"
open ios/Runner.xcworkspace
```

### 步骤 2：连接设备并运行

1. 在 Xcode 中选择设备：`Dianhua`
2. 点击运行按钮（或按 `Cmd + R`）
3. 应用启动后，停止运行（应用会留在手机上）

### 步骤 3：查看 Console

1. 在 Xcode 底部打开 **Console** 窗口
2. **重启手机** → 解锁
3. **直接点击桌面上的应用图标**
4. **观察 Console 输出**：如果崩溃，会显示详细的崩溃信息

---
