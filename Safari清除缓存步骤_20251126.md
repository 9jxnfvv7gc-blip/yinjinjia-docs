# 📱 Safari 清除缓存步骤（2025-11-26）

## ✅ 清除 Safari 历史记录和网站数据

### 方法1：通过设置清除（推荐）

1. **打开"设置"应用**
2. **向下滚动，找到"Safari"**（在应用列表中）
3. **点击"Safari"**
4. **向下滚动，找到"清除历史记录与网站数据"**
5. **点击"清除历史记录与网站数据"**
6. **确认清除**

### 方法2：在 Safari 中清除

1. **打开 Safari 浏览器**
2. **点击右下角的"书签"图标**（打开的书本图标）
3. **点击"历史记录"标签**（时钟图标）
4. **点击左下角的"清除"**
5. **选择"全部时间"**
6. **点击"清除历史记录"**

---

## 🔍 如果找不到"清除历史记录与网站数据"

### 可能的位置：

1. **设置 → Safari → 清除历史记录与网站数据**
   - 这个选项在 Safari 设置页面的底部
   - 可能需要向下滚动

2. **或者**：
   - 设置 → Safari → 高级 → 网站数据 → 清除所有网站数据

---

## ✅ 推荐操作

### 最简单的方法：

1. **设置 → Safari**
2. **向下滚动到底部**
3. **找到"清除历史记录与网站数据"**
4. **点击并确认**

---

## 📝 清除后的操作

清除缓存后：

1. **重新连接 Wi-Fi**（确保代理配置还在）
2. **打开 Safari**
3. **访问**：`http://www.baidu.com`（先测试代理是否工作）
4. **如果能打开**，再访问：`https://accounts.google.com/signup`
