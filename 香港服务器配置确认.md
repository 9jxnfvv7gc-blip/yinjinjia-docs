### 如果服务器无法访问

需要启动香港服务器的 `video_server.py`：

1. **连接到香港服务器**：
   ```bash
   ssh root@47.243.177.166
   ```

2. **启动服务器**：
   ```bash
   export VIDEO_ROOT="/root/videos"
   cd /root/app  # 或 video_server.py 所在目录
   nohup python3 video_server.py > /root/server.log 2>&1 &
   ps aux | grep video_server.py
   ```
