# 如何找到"优途漫游畅游世界"的代理信息

## 方法 1：在应用内查找
1. 打开"优途漫游畅游世界"应用
2. 查找以下位置：
   - **设置** 或 **配置**
   - **代理设置**
   - **系统代理**
   - **本地代理端口**
   - **HTTP代理端口**

## 方法 2：查看应用界面
通常会在以下位置显示：
- 主界面底部
- 设置页面
- 关于页面
- 帮助文档

## 方法 3：常见端口
如果找不到，可以尝试这些常见端口：
- `7890`（Clash 默认）
- `1080`（Shadowsocks 默认）
- `1087`（V2Ray 默认）
- `8080`（常见 HTTP 代理）
- `8888`（常见 HTTP 代理）

## 方法 4：测试连接
1. 启动"优途漫游畅游世界"
2. 连接到一个节点
3. 打开浏览器，访问 `https://www.google.com`
4. 如果能访问，说明代理已启用

## 代理地址
通常代理地址是：
- `127.0.0.1` 或
- `localhost`
