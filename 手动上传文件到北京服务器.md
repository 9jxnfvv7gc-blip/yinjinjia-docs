# 手动上传文件到北京服务器

## 📤 方法1：在你的Mac终端手动执行scp

### 步骤1：打开Mac终端（Terminal.app）

不要使用Cursor的终端，使用系统的Terminal.app。

### 步骤2：执行scp命令

```bash
scp "/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905/video_server.py" admin@39.107.137.136:/tmp/
```

**如果提示输入密码**：
- 输入admin用户的密码
- 密码输入时不会显示（这是正常的）
- 输入完成后按回车

### 步骤3：在服务器上检查

在服务器终端执行：

```bash
ls -la /tmp/video_server.py
```

如果文件存在，继续下一步。

## 📤 方法2：使用Workbench的"上传文件"功能

### 步骤1：在Workbench中查找上传功能

1. **查看Workbench界面**：
   - 顶部菜单栏是否有"文件"或"上传"选项
   - 右键点击文件列表区域，看是否有"上传"选项
   - 查看是否有"+"或"上传"按钮

2. **如果找到上传功能**：
   - 选择 `/home/admin/` 或 `/tmp/` 目录
   - 上传 `video_server.py` 文件
