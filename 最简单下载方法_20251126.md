# 🎯 最简单的下载方法（2025-11-26）

## ❌ 问题

通过命令行下载遇到 404 错误，版本号可能不对。

---

## ✅ 推荐方案：在远程桌面查看并下载

这是最可靠的方法，可以直接看到正确的下载链接。

---

## 🚀 操作步骤

### 步骤1：连接远程桌面

1. **打开 Windows App**（远程桌面应用）
2. **连接到服务器**：
   - 服务器：`47.243.177.166`
   - 用户名：`desktop`
   - 密码：你的密码

### 步骤2：在远程桌面中下载

1. **打开 Firefox**（在远程桌面中）：
   - 如果 Firefox 没有运行，在终端运行：`firefox &`

2. **访问 GitHub releases 页面**：
   - **Clash for Windows**：https://github.com/Fndroid/clash_for_windows_pkg/releases
   - **或 ClashX Pro**：https://github.com/yichengchen/clashX/releases

3. **找到最新版本**：
   - 页面会显示所有版本
   - 找到最新版本（通常显示 "Latest" 标签）
   - 向下滚动，找到 "Assets" 部分

4. **找到 Mac 版本的 .dmg 文件**：
   - 查找文件名包含 `.dmg` 的文件
   - 对于 Clash for Windows，查找 `Clash.for.Windows-*-x64.dmg` 或 `Clash.for.Windows-*-mac.dmg`
   - 对于 ClashX Pro，查找 `ClashX-Pro-*.dmg`

5. **下载文件**：
   - 点击 .dmg 文件，开始下载
   - 等待下载完成（文件通常下载到 `~/Downloads/` 目录）

### 步骤3：传输到 Mac

**退出远程桌面**，在 Mac 终端运行：

```bash
# 传输文件到 Mac
scp desktop@47.243.177.166:~/Downloads/*.dmg ~/Downloads/
```

如果提示输入密码，输入 `desktop` 用户的密码。

## 🔍 如果找不到下载链接

### 方法1：搜索其他代理客户端

可以尝试搜索其他 Mac 代理客户端：

1. **V2RayU**：
   - https://github.com/yanue/V2rayU/releases
   - 查找 `.dmg` 文件

2. **ShadowsocksX-NG**：
   - https://github.com/shadowsocks/ShadowsocksX-NG/releases
   - 查找 `.dmg` 文件

3. **Clash Verge**：
   - https://github.com/zzzgydi/clash-verge/releases
   - 查找 Mac 版本的 `.dmg` 文件

### 方法2：使用命令行工具（3Proxy）

如果图形界面工具都下载困难，可以使用命令行工具：

```bash
# 在 Mac 上安装 Homebrew（如果还没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 3Proxy
brew install 3proxy
```

然后配置 3Proxy 作为代理转发工具。
