# 🔍 查找视频文件位置

## 📋 在服务器终端执行

### 方法1：查找所有视频文件

```bash
find /root -type f \( -name "*.mp4" -o -name "*.mov" -o -name "*.mkv" -o -name "*.avi" \) 2>/dev/null | head -10
```

### 方法2：查找包含"视频"的目录

```bash
find /root -type d -name "*视频*" 2>/dev/null
```

### 方法3：查看 /root 目录内容

```bash
ls -la /root/
```

### 方法4：检查其他常见位置

```bash
ls -la /home/
ls -la /var/www/
ls -la /opt/
```

## 🎯 找到视频文件后

如果找到了视频文件，例如在 `/root/my_videos/` 目录：

```bash
# 修改 VIDEO_ROOT 环境变量
export VIDEO_ROOT="/root/my_videos"

# 停止旧服务器
pkill -f video_server.py

# 重新启动服务器
cd /root/app
nohup python3 video_server.py > /root/server.log 2>&1 &

# 测试
curl http://localhost:8081/api/list/原创视频
```
