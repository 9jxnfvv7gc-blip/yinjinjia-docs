# 当前问题排查和解决方案

## 问题 1：协议链接显示乱码

### 可能原因
- 应用没有热重载，还在使用旧版本（外部链接）
- 或者协议页面渲染有问题

### 解决方案
1. **完全重启应用**（不是热重载）：
   ```bash
   # 停止当前应用（在终端按 q）
   # 然后重新运行
   flutter run -d Dianhua
   ```

2. **检查协议显示**：
   - 点击设置 → 用户协议
   - 应该看到应用内页面，不是外部浏览器
   - 如果还是乱码，可能是渲染问题
