# 🔧 Clash Verge 无法访问 Google 最终排查（2025-11-26）

## ❌ 问题

配置完成后，还是打不开 Google。

---

## 🔍 深度排查

### 步骤1：检查系统代理是否真的设置了

在终端运行：

```bash
networksetup -getwebproxy "Wi-Fi"
```

如果显示 "Enabled: No"，说明系统代理没有设置。

### 步骤2：手动设置系统代理

如果系统代理没有自动设置，需要手动设置：

1. **系统设置 → 网络**
2. **选择当前网络连接**（Wi-Fi 或以太网）
3. **点击"高级"**
4. **切换到"代理"标签**
5. **勾选"网页代理（HTTP）"**
6. **填写**：
   - 服务器：127.0.0.1
   - 端口：7890
7. **勾选"安全网页代理（HTTPS）"**（如果有）
8. **填写**：
   - 服务器：127.0.0.1
   - 端口：7890
9. **保存**

### 步骤3：测试本地代理

在终端运行：

```bash
curl -v --proxy http://127.0.0.1:7890 https://www.google.com
```

如果返回内容，说明本地代理工作正常。

## ✅ 替代方案

### 方案1：使用 SOCKS5 代理（推荐）

如果 HTTP 代理不工作，尝试 SOCKS5：

1. **系统设置 → 网络 → 高级 → 代理**
2. **取消勾选"网页代理（HTTP）"**
3. **勾选"SOCKS 代理"**
4. **填写**：
   - 服务器：127.0.0.1
   - 端口：7891
5. **保存**
6. **测试访问 Google**

既然 Clash Verge 配置复杂，建议：

1. **使用远程桌面**（Windows App）

3. **完成 Google 注册**

这是最简单、最可靠的方法。
