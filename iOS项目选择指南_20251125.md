## 🐛 如果运行失败（Failed）

### 查看错误信息

Xcode 底部会显示错误信息，常见错误：

#### 错误1：签名失败（Code Signing Error）
```
Code signing is required for product type 'Application' in SDK 'iOS'
```
**解决**：
- 在 Signing & Capabilities 中选择正确的 Team
- 确保 Bundle ID 唯一
- 可能需要修改 Bundle ID 为：`com.xiaohui.videomusicapp`

#### 错误2：设备未连接
```
No devices found
```
**解决**：
- 确保 iPhone 用 USB 连接
- 在 iPhone 上点击"信任此电脑"
- 在 Xcode 的设备选择器中刷新

#### 错误3：依赖问题
```
Pod installation failed
```
**解决**：
```bash
cd ios
pod deintegrate
pod install
cd ..
flutter clean
flutter pub get
```

#### 错误4：编译错误
查看 Xcode 底部的具体错误信息，通常会有：
- 红色错误标记
- 黄色警告（可能不影响运行）
