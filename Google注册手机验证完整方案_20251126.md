# 📱 Google 注册手机验证完整方案（2025-11-26）

## ❌ 问题

在远程桌面申请 Google 时，手机验证是必经的一步。但手机无法访问 Google。

---

## ✅ 解决方案

### 方案1：让 iPhone 通过 Mac 的 Clash Verge 访问 Google（推荐）

既然我们已经配置了 Clash Verge，可以：

#### 步骤1：修复 Clash Verge 的 Mac 访问问题

**可能的问题**：
- Clash Verge 的 HTTPS 处理有问题
- 需要重新启动或重新配置

**尝试方法**：
1. **完全退出 Clash Verge**
2. **重新启动 Clash Verge**
3. **选择 `hongkong` 配置**
4. **选择 "Proxy" 代理组**
5. **开启系统代理**
6. **测试 Mac 访问 Google**

#### 步骤2：开启局域网共享

1. **打开 Clash Verge 设置**
2. **找到"允许局域网连接"或"Allow LAN"**
3. **✅ 勾选这个选项**
4. **记录 HTTP 端口**：7890

#### 步骤3：配置 iPhone Wi-Fi 代理

1. **查看 Mac 内网 IP**：
   ```bash
   ipconfig getifaddr en0
   ```

2. **在 iPhone 上配置 Wi-Fi 代理**：
   - 设置 → Wi-Fi → 点击当前网络 ⓘ
   - HTTP 代理 → 手动
   - 服务器：Mac 的内网 IP
   - 端口：7890
   - 保存

3. **测试 iPhone 访问 Google**：
   - 访问：`https://www.google.com`
   - 如果能看到 Google，说明成功 ✅

#### 步骤4：在远程桌面完成注册，用 iPhone 验证

1. **在远程桌面 Firefox 中访问**：`https://accounts.google.com/signup`
2. **填写注册信息**
3. **当提示手机验证时**：
   - 用 iPhone 扫描二维码
   - iPhone 应该能访问 Google 验证页面（通过 Mac 的 Clash Verge）
   - 完成验证
