# 使用Workbench文件管理器上传内容

## 📁 目标目录

服务器上的目标目录：
- `/root/videos/原创视频/` - 存放视频文件
- `/root/videos/原创歌曲/` - 存放歌曲文件

---

## 🚀 操作步骤

### 步骤1：打开Workbench文件管理器

1. 登录阿里云控制台
2. 进入 **ECS实例** → 选择你的服务器（IP: 47.243.177.166）
3. 点击 **远程连接** → **Workbench远程连接**
4. 点击左侧的 **文件管理** 图标（文件夹图标）

---

### 步骤2：导航到目标目录

在文件管理器中：

1. 在左侧目录树中，导航到 `/root/videos/`
2. 如果看不到 `/root/videos/` 目录，可以：
   - 在地址栏输入：`/root/videos`
   - 或者点击 **根目录** → `root` → `videos`

---

### 步骤3：上传视频文件

1. 进入 `/root/videos/原创视频/` 目录
2. 点击 **上传** 按钮（通常在顶部工具栏）
3. 选择 **上传文件** 或 **上传文件夹**
4. 从你的Mac电脑选择视频文件：
   - 本地路径：`/Volumes/Expansion/原创视频/`
   - 选择要上传的视频文件
5. 等待上传完成

---

### 步骤4：上传歌曲文件

1. 进入 `/root/videos/原创歌曲/` 目录
2. 点击 **上传** 按钮
3. 选择 **上传文件** 或 **上传文件夹**
4. 从你的Mac电脑选择歌曲文件：
   - 本地路径：`/Volumes/Expansion/原创歌曲/`
   - 选择要上传的歌曲文件
5. 等待上传完成

---

## 📋 详细操作指南

### 方法1：单个文件上传

1. 打开Workbench文件管理器
2. 导航到 `/root/videos/原创视频/` 或 `/root/videos/原创歌曲/`
3. 点击 **上传** 按钮
4. 选择本地文件
5. 等待上传完成

---

### 方法2：批量上传（文件夹）

1. 打开Workbench文件管理器
2. 导航到 `/root/videos/`
3. 点击 **上传** → **上传文件夹**
4. 选择本地文件夹（如 `/Volumes/Expansion/原创视频/`）
5. 等待上传完成

**注意**：上传文件夹时，会保持文件夹结构。

---

### 方法3：拖拽上传

1. 打开Workbench文件管理器
2. 导航到目标目录
3. 从Mac的Finder中拖拽文件到Workbench文件管理器
4. 等待上传完成

---

## 💡 提示

1. **文件大小限制**：Workbench文件管理器对单个文件大小可能有限制（通常几GB），如果文件很大，建议使用 `rsync` 或分块上传。

2. **上传速度**：取决于你的网络速度和文件大小，大文件可能需要较长时间。

3. **文件格式**：
   - 视频：`.mp4`, `.mov`, `.avi` 等
   - 歌曲：`.mp3`, `.m4a`, `.wav` 等

4. **文件命名**：建议使用中文文件名，服务器已支持UTF-8编码。
