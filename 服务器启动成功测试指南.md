# 服务器启动成功测试指南

## ✅ 服务器状态

- **状态**：✅ 正在运行
- **端口**：8081
- **视频根目录**：/Volumes/Expansion
- **服务器地址**：http://192.168.10.103:8081

## 🧪 测试步骤

### 1. 在 Mac 上测试服务器

在终端运行：
```bash
curl http://192.168.10.103:8081/api/list/原创视频
curl http://192.168.10.103:8081/api/list/原创歌曲
```

如果返回 JSON 数据（即使是空数组 `[]`），说明服务器正常。

### 2. 在 iOS 设备 Safari 中测试

1. 确保 iOS 设备和 Mac 在同一 WiFi 网络
2. 在 Safari 中访问：
   ```
   http://192.168.10.103:8081/api/list/原创视频
   ```
3. 如果能看到 JSON 数据，说明网络正常

### 3. 运行应用测试

```bash
cd "/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905"
flutter run -d Dianhua
```

应用应该能够：
- 连接到服务器
- 加载视频和音乐列表
- 显示内容（如果有的话）

## 📝 注意事项

1. **保持服务器运行**：启动服务器的终端窗口必须保持打开
2. **如果目录为空**：服务器会返回空数组 `[]`，这是正常的
3. **网络要求**：Mac 和 iOS 设备必须在同一 WiFi 网络

## 🔄 如果仍然无法连接

1. **检查 WiFi**：确保 Mac 和 iOS 设备在同一网络
2. **检查防火墙**：Mac 防火墙可能阻止连接
3. **检查 IP 地址**：如果 Mac IP 变化，需要更新 `lib/config.dart`
