# 🔐 Xcode 签名配置详细步骤（2025-11-25）

## 📍 如何找到 Signing & Capabilities

### 方法1：通过项目设置（推荐）

1. **在 Xcode 左侧项目导航器中**：
   - 点击最顶部的蓝色图标（项目名称，通常是 `Runner`）
   - **不是**点击文件夹，而是点击项目本身

2. **在中间的主编辑区域**：
   - 应该看到 `TARGETS` 和 `PROJECT` 两个部分
   - 在 `TARGETS` 下，点击 `Runner`（不是 RunnerTests）

3. **在顶部标签栏**：
   - 应该看到：`General`、`Signing & Capabilities`、`Resource Tags` 等标签
   - 点击 `Signing & Capabilities` 标签

4. **在 Signing & Capabilities 页面**：
   - 找到 `Signing` 部分
   - 在 `Team` 下拉菜单中选择你的 Apple ID

### 方法2：如果看不到 Signing & Capabilities 标签

#### 检查是否选对了 target

1. **确认选择的是 Runner target**：
   - 在左侧项目导航器，点击项目（蓝色图标）
   - 在中间区域，确认 `TARGETS` 下的 `Runner` 被选中（高亮显示）
   - **不要**选择 `RunnerTests`

2. **检查 Xcode 版本**：
   - 在 Xcode 菜单：`Xcode` → `About Xcode`
   - 如果是旧版本，界面可能不同

#### 替代方法：通过 Build Settings

1. **选择 Runner target**（同上）

2. **点击 `Build Settings` 标签**（在顶部标签栏）

3. **搜索 "signing"**：
   - 在右上角的搜索框输入 `signing`
   - 找到 `Code Signing Identity`
   - 找到 `Development Team`

4. **设置 Development Team**：
   - 点击 `Development Team` 右侧的值
   - 选择你的 Apple ID（`lmminakols@hotmail.com`）

## 🔍 详细步骤（带截图说明）

### 步骤1：确认项目结构

在 Xcode 左侧，应该看到：
```
Runner (蓝色图标，这是项目)
  ├── Runner (这是 target)
  ├── RunnerTests
  └── Pods
```

### 步骤2：选择正确的 target

1. 点击左侧的 `Runner`（蓝色项目图标）
2. 在中间区域，找到 `TARGETS` 部分
3. 点击 `Runner`（确保它被选中，背景高亮）

### 步骤3：查找 Signing 选项

**如果看到 `Signing & Capabilities` 标签**：
- 直接点击这个标签
- 找到 `Team` 下拉菜单

**如果只看到 `General` 标签**：
- 点击 `General` 标签
- 向下滚动，找到 `Signing` 部分
- 在 `Team` 下拉菜单中选择你的 Apple ID

**如果看到 `Build Settings` 标签**：
- 点击 `Build Settings` 标签
- 搜索 `signing` 或 `team`
- 找到 `Development Team` 并设置

## 🆘 如果仍然找不到

### 方法1：检查 Xcode 窗口大小

- 标签栏可能被隐藏了
- 尝试调整 Xcode 窗口大小
- 或者点击 `View` → `Navigators` → `Show Project Navigator`

### 方法2：使用菜单

1. 选择 `Runner` target
2. 在 Xcode 顶部菜单：`Editor` → `Add Capability...`
3. 或者：`File` → `Project Settings...` → `Signing`

### 方法3：直接编辑项目文件（不推荐，但可行）

如果以上方法都不行，可以告诉我，我会帮你直接修改项目配置文件。
