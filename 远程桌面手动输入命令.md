# 🖥️ 远程桌面手动输入命令（一条一条输入）

## 📋 说明
- 每次只输入一条命令
- 按 Enter 执行
- 等待执行完成后再输入下一条
- 如果提示输入密码，输入 root 密码

---

## 🔤 第一步：安装中文字体（解决乱码）

### 命令1：更新软件包列表
```
sudo apt update
```
**说明**：等待执行完成，可能需要几秒钟

---

### 命令2：安装中文字体包1
```
sudo apt install fonts-wqy-zenhei -y
```
**说明**：等待安装完成，输入 y 确认（如果有提示）

---

### 命令3：安装中文字体包2
```
sudo apt install fonts-noto-cjk -y
```
**说明**：等待安装完成

---

### 命令4：刷新字体缓存
```
sudo fc-cache -fv
```
**说明**：让系统识别新安装的字体

---

## 🌐 第二步：打开 Firefox（如果还没打开）

### 命令5：打开 Firefox
```
firefox
```
**说明**：Firefox 会在后台打开，终端可以继续使用

---

## 🔍 第三步：检查字体是否安装成功（可选）

### 命令6：查看中文字体
```
fc-list :lang=zh | head -5
```
**说明**：如果看到字体列表，说明安装成功

---

### 安装字体部分：
```
sudo apt update
```

```
sudo apt install fonts-wqy-zenhei -y
```

```
sudo apt install fonts-noto-cjk -y

```
sudo fc-cache -fv

### 打开 Firefox：
```
firefox
```

---

## 🌐 安装翻译扩展（在 Firefox 中操作）

### 步骤1：在 Firefox 地址栏输入
```
https://addons.mozilla.org/zh-CN/firefox/addon/google-translator-for-firefox/
```

### 步骤2：点击"添加到 Firefox"按钮

### 步骤3：确认安装

---

## ⚡ 如果远程桌面很卡，优化性能

### 命令：查看占用资源的进程
```
top
```
**说明**：按 q 退出

### 命令：杀掉 Gradle 进程（如果占用过多）
```
pkill -f gradle
```

### 命令：查看内存使用
```
free -h
```

---

## ✅ 验证安装

### 命令：重启 Firefox（让字体生效）
```
pkill firefox
```

```
firefox
```

---

## 📋 最简化的命令列表（最少步骤）

如果只想快速安装，只执行这3条：

```
sudo apt update
```

```
sudo apt install fonts-wqy-zenhei fonts-noto-cjk -y
```

```
sudo fc-cache -fv
```

然后打开 Firefox，安装翻译扩展。

---

## 💡 提示

- 每条命令执行完后再输入下一条
- 如果提示输入密码，输入 root 密码
- 如果提示确认，输入 y 然后按 Enter
- 安装字体后需要重启 Firefox 才能生效
