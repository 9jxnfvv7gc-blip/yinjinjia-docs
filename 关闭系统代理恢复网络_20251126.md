# 🔧 关闭系统代理恢复网络（2025-11-26）

## ❌ 问题

退出了 Clash Verge，但还是打不开豆包。说明系统代理可能还在开启状态。

---

## ✅ 解决方案

### 方法1：手动关闭系统代理（推荐）

1. **系统设置 → 网络**
2. **选择当前网络连接**（Wi-Fi 或以太网）
3. **点击"高级"**
4. **切换到"代理"标签**
5. **取消勾选所有代理选项**：
   - 取消勾选"网页代理（HTTP）"
   - 取消勾选"安全网页代理（HTTPS）"（如果有）
   - 取消勾选"SOCKS 代理"（如果有）
6. **保存**

### 方法2：使用终端关闭（如果方法1不行）

在终端运行：

```bash
# 关闭 HTTP 代理
networksetup -setwebproxystate "Wi-Fi" off

# 关闭 SOCKS 代理
networksetup -setsocksfirewallproxystate "Wi-Fi" off
```

如果使用的是以太网，将 "Wi-Fi" 改为 "Ethernet"。

---

## 🔍 验证代理已关闭

在终端运行：

```bash
networksetup -getwebproxy "Wi-Fi"
```

如果显示 "Enabled: No"，说明代理已关闭。

---

## 📝 如果还是打不开

### 检查其他可能的问题

1. **检查网络连接**：
   - 测试访问其他网站（如 baidu.com）
   - 如果其他网站也打不开，可能是网络问题

2. **清除浏览器缓存**：
   - Safari → 设置 → 隐私 → 清除历史记录和网站数据

3. **重新连接 Wi-Fi**：
   - 关闭 Wi-Fi，等待 5 秒，重新打开
