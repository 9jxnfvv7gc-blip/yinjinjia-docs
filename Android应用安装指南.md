## 🔌 方法1: 通过USB连接安装（推荐）

### 步骤1: 连接手机
1. 用USB线连接Android手机到Mac
2. 在手机上启用"USB调试"：
   - 设置 → 关于手机 → 连续点击"版本号"7次（开启开发者模式）
   - 设置 → 系统 → 开发者选项 → 开启"USB调试"
3. 手机上点击"信任此电脑"

### 步骤2: 检查连接
在Mac终端运行：
```bash
adb devices
```
如果显示设备ID，说明连接成功。

### 步骤3: 安装APK
```bash
adb install -r ~/Desktop/app-release-new.apk
```

## 📲 方法2: 直接传输安装（无需USB）

### 步骤1: 传输APK到手机
- **AirDrop**（如果手机支持）
- **微信文件传输助手**
- **QQ文件传输**
- **邮件发送给自己**

### 步骤2: 在手机上安装
1. 打开文件管理器
2. 找到下载的APK文件
3. 点击安装
4. 如果提示"未知来源"，需要：
   - 设置 → 安全 → 允许安装未知来源应用

## 🧪 测试应用

### 安装后检查：
1. ✅ 应用是否能正常打开
2. ✅ 是否能显示视频列表
3. ✅ 视频是否能播放

### 如果看不到视频：
1. **检查网络连接**：
   - 在手机浏览器访问：`http://47.243.177.166:8081/`
   - 如果能看到服务器状态，说明网络正常

2. **查看应用日志**（需要USB连接）：
   ```bash
   adb logcat -c
   # 在手机上打开应用
   adb logcat | grep -i "error\|exception\|http" --line-buffered
   ```

3. **清除应用数据**：
   - 设置 → 应用 → 影音播放器 → 存储 → 清除数据

## 🔍 常见问题

### Q: adb devices 显示空列表？
**A**: 
- 检查USB连接
- 确认已启用USB调试
- 尝试更换USB线或USB端口
- 在手机上点击"信任此电脑"

### Q: 安装失败？
**A**:
- 先卸载旧版本：`adb uninstall com.example.videoMusicApp`
- 然后重新安装

### Q: 应用看不到视频？
**A**:
- 检查手机网络连接
- 在浏览器测试服务器是否可访问
- 查看应用日志找出错误
