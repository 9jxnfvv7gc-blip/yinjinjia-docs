# 📱 Android网络权限手动授予指南

## 🔧 问题说明

当前安装的APK缺少网络权限配置，导致无法访问服务器获取视频列表。

## ✅ 解决方案：手动授予网络权限

### 步骤1：打开应用设置

1. **打开手机设置**
2. **找到"应用"或"应用管理"**
3. **找到"影音播放器"应用**
4. **点击进入应用详情**

### 步骤2：授予网络权限

1. **点击"权限"**
2. **找到"网络"或"互联网"权限**
3. **设置为"允许"**

### 步骤3：检查其他权限

确保以下权限已授予：
- ✅ **网络/互联网** - 必须
- ✅ **存储**（如果需要上传文件）

### 步骤4：重新打开应用

1. **返回桌面**
2. **重新打开"影音播放器"应用**
3. **检查是否能显示视频列表**

## 🔍 不同手机品牌的设置路径

### 小米/红米
- 设置 → 应用 → 应用管理 → 影音播放器 → 权限管理 → 网络

### OPPO/一加
- 设置 → 应用 → 应用管理 → 影音播放器 → 权限 → 网络

### vivo
- 设置 → 应用与权限 → 应用管理 → 影音播放器 → 权限 → 网络

### 华为/荣耀
- 设置 → 应用 → 应用管理 → 影音播放器 → 权限 → 网络

### 通用方法
- 设置 → 应用 → 找到"影音播放器" → 权限 → 网络

## ⚠️ 如果还是不行

如果手动授予权限后还是看不到视频：

1. **检查网络连接**：
   - 确保手机连接到WiFi或移动网络
   - 尝试在浏览器中访问：`http://47.243.177.166:8081`

2. **清除应用数据**：
   - 设置 → 应用 → 影音播放器 → 存储 → 清除数据
   - 重新打开应用

3. **重新安装应用**：
   - 等网络恢复后，重新编译安装新版本（已修复网络权限）

## 📝 已修复的配置

我已经在代码中添加了：
- ✅ `INTERNET` 权限
- ✅ `ACCESS_NETWORK_STATE` 权限  
- ✅ `usesCleartextTraffic="true"`（允许HTTP连接）

等网络恢复后，重新编译安装新版本即可。
