我已经在浏览器中打开了 ClashX 的下载页面。

1. **在浏览器中找到 ClashX 的下载页面**（应该已经自动打开了）

2. **找到最新的版本**（例如：`1.95.1` 或更高版本）

3. **点击下载 `ClashX.dmg` 文件**

### 方法2：使用 Homebrew 安装（如果已安装 Homebrew）


### 方法3：手动下载链接

如果浏览器页面没有自动打开，请手动访问：

```
https://github.com/yichengchen/clashX/releases
```

然后：
1. 找到最新的 release（通常在最上面）
2. 找到 `Assets` 部分
3. 点击下载 `ClashX.dmg` 文件
