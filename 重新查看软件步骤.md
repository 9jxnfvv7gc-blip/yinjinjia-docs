## 🔧 常见问题

### 问题1：服务器连接失败

**错误信息**：`Connection refused` 或 `加载失败`

**解决方法**：
1. 确认 `video_server.py` 正在运行
2. 检查终端是否显示 "Serving at port 8081"
3. 重新启动服务器

---

### 问题2：没有内容显示

**可能原因**：
- `/Volumes/Expansion/原创视频` 或 `/Volumes/Expansion/原创歌曲` 目录为空
- 服务器未正确启动

**解决方法**：
1. 检查目录是否有文件
2. 上传一些测试内容
3. 重新加载应用（下拉刷新）

---

### 问题3：应用窗口没有显示（macOS）

**解决方法**：
1. 检查Dock中是否有应用图标
2. 使用Command+Tab切换应用
3. 使用Mission Control查看所有窗口
4. 重新运行 `flutter run -d macos`

---

### 问题4：iOS模拟器黑屏

**解决方法**：
1. 关闭模拟器
2. 重新运行 `flutter run -d ios`
3. 等待应用完全加载
