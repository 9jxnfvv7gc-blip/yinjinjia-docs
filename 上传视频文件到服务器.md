# 📤 上传视频文件到服务器

## 📋 当前状态

- ✅ 服务器已启动并运行正常
- ✅ 目录结构正确：`/root/videos/原创视频/` 和 `/root/videos/原创歌曲/`
- ❌ 视频目录为空，需要上传视频文件

## 🎯 上传视频文件

### 方法1：从本地 Mac 上传（如果视频在本地）

在本地 Mac 终端执行：

```bash
# 上传单个视频文件
scp /本地视频路径/视频.mp4 admin@39.107.137.136:/tmp/

# 上传整个目录
scp -r /本地视频目录/* admin@39.107.137.136:/tmp/videos/
```

然后在服务器上移动：

```bash
sudo mv /tmp/视频.mp4 /root/videos/原创视频/
# 或
sudo mv /tmp/videos/* /root/videos/原创视频/
```

### 方法2：从香港服务器复制（如果视频在香港服务器）

在服务器上执行：

```bash
# 从香港服务器复制视频
scp root@47.243.177.166:/root/videos/原创视频/* /root/videos/原创视频/
```

### 方法3：使用阿里云文件管理

1. 登录阿里云控制台
2. 进入 ECS → 文件管理
3. 上传视频文件到 `/root/videos/原创视频/` 目录

## ✅ 上传后测试

上传视频文件后，在服务器上执行：

```bash
# 检查文件
ls -la /root/videos/原创视频/

# 测试 API
curl http://localhost:8081/api/list/原创视频

# 应该返回视频列表 JSON
```

## 📝 视频文件要求

- 支持的格式：.mp4, .mov, .mkv, .avi 等
- 文件放在：`/root/videos/原创视频/` 目录
- 音乐文件放在：`/root/videos/原创歌曲/` 目录

## 🚀 上传完成后

1. 在 iPhone 上刷新应用
2. 应该能看到视频列表了
