# 📱 如何清理 iOS 分析与改进数据

## 🎯 目的

清理旧的崩溃日志，以便看到最新的崩溃日志。

## 📋 步骤

### 方法 1：删除特定应用的日志（推荐）

1. **打开设置**：`设置 → 隐私与安全性 → 分析与改进 → 分析数据`
2. **找到应用日志**：查找所有 `Runner-...` 或 `com.xiaohui.videoMusicApp-...` 的日志
3. **删除日志**：
   - 点击进入每个日志
   - 点击右上角的 **"分享"** 按钮
   - 选择 **"删除"**
   - 或者直接左滑日志条目，点击 **"删除"**

### 方法 2：删除所有分析数据（不推荐）

1. **打开设置**：`设置 → 隐私与安全性 → 分析与改进`
2. **关闭分析**：关闭 **"共享 iPhone 分析"** 和 **"共享 iCloud 分析"**
3. **等待几秒**：系统会清理数据
4. **重新开启**：重新打开这些选项

### 方法 3：重启手机后查看

重启手机后，新的崩溃日志会出现在列表最前面（按时间排序）。

## ⚠️ 注意事项

- 删除日志不会影响应用功能
- 删除后，新的崩溃日志会在下次崩溃时自动生成
- 建议只删除旧的日志，保留最新的

## 🔍 获取最新日志

清理后，如果应用再次崩溃：
1. 打开：`设置 → 隐私与安全性 → 分析与改进 → 分析数据`
2. 找到**时间最新**的日志（刚刚崩溃的时间）
3. 点击进入，全选并复制完整内容
