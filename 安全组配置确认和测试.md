## 🔍 测试连接

### 步骤1：从本地Mac测试服务器

在本地 Mac 终端执行：

```bash
curl http://47.243.177.166:8081/api/list/%E5%8E%9F%E5%88%9B%E8%A7%86%E9%A2%91
```

如果返回包含2个视频的JSON数组，说明服务器可以访问！

### 步骤2：如果仍然无法访问

检查服务器防火墙：

在服务器终端执行：

```bash
# 检查防火墙状态
sudo ufw status

# 如果防火墙阻止，开放端口
sudo ufw allow 8081/tcp
sudo ufw reload
```

### 步骤3：运行应用

服务器可以访问后，在本地 Mac 终端执行：

```bash
cd "/Volumes/Expansion/FlutterProjects/桌面影音播放器_安装包_20251121_165905"
flutter run -d Dianhua
```

删除其他规则是安全的，只要：

1. 防火墙阻止（需要在服务器上开放）
2. 应用代码需要重新编译
3. 网络问题
