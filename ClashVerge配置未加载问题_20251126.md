# 🔧 Clash Verge 配置未加载问题（2025-11-26）

## ❌ 问题

在 Proxies 标签中看不到 "Proxy" 或 "HongKong-Squid"，只看到 rule, global, direct, script。

这说明配置文件可能没有正确加载。

---

## ✅ 解决方案

### 方法1：重新加载配置

1. **在 Clash Verge 中**
2. **切换到 Profiles 标签**
3. **找到 `hongkong` 配置**
4. **右键点击或点击配置名称旁边的菜单**
5. **选择"重新加载"、"Reload"或"刷新"**
6. **等待几秒，让配置加载**

### 方法2：重新选择配置

1. **在 Profiles 标签中**
2. **取消选择当前配置**（如果有）
3. **重新选择 `hongkong` 配置**
4. **等待配置加载**

### 方法3：重新启动 Clash Verge

1. **完全退出 Clash Verge**（右键点击菜单栏图标 → 退出）
2. **重新启动 Clash Verge**
3. **在 Profiles 标签中选择 `hongkong` 配置**
4. **等待配置加载**

---

## 🔍 检查配置文件

### 确认配置文件内容正确

配置文件应该包含：
- `proxies`：代理服务器列表（应该有 "HongKong-Squid"）
- `proxy-groups`：代理组列表（应该有 "Proxy"）
- `rules`：规则列表

如果配置文件内容不正确，需要重新创建。
