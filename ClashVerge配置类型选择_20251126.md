# 📝 Clash Verge 配置类型选择（2025-11-26）

## 🎯 选择配置类型

点击 "new" 后，出现了类型选择：
- **remote**：从远程 URL 导入配置
- **local**：本地配置文件 ✅ **选择这个**
- **script**：脚本配置
- **merge**：合并配置

---

## ✅ 选择 "local"

### 为什么选择 "local"？

1. **我们要手动创建配置文件**
2. **配置文件存储在本地**
3. **不需要从远程 URL 导入**
4. **不需要脚本或合并功能**

---

## 📝 操作步骤

### 步骤1：选择 "local"

1. **点击 "new" 按钮**
2. **在类型选择中，选择 "local"**
3. **点击"下一步"或"创建"**

### 步骤2：输入配置名称

1. **会要求输入配置名称**
2. **输入**：`hongkong`
3. **确认或点击"创建"**

### 步骤3：编辑配置文件

1. **会打开一个编辑器窗口**
2. **清空默认内容**（如果有）
3. **粘贴以下配置内容**：

```yaml
port: 7890
socks-port: 7891
allow-lan: true
mode: rule
log-level: info

proxies:
  - name: "HongKong-Squid"
    type: http
    server: 47.243.177.166
    port: 3128

proxy-groups:
  - name: "Proxy"
    type: select
    proxies:
      - "HongKong-Squid"

rules:
  - MATCH,Proxy
```

4. **保存配置文件**（点击"保存"按钮或按 Command + S）

---

## 🔍 各类型说明

### remote（远程）

- **用途**：从 URL 导入配置
- **我们不需要**：因为我们手动创建配置

### local（本地）✅ **选择这个**

- **用途**：创建本地配置文件
- **我们使用**：手动创建和编辑配置

### script（脚本）

- **用途**：使用脚本生成配置
- **我们不需要**：配置很简单，不需要脚本

### merge（合并）

- **用途**：合并多个配置文件
- **我们不需要**：只有一个配置
