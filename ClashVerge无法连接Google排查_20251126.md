# 🔧 Clash Verge 无法连接 Google 排查（2025-11-26）

## ❌ 问题

Safari 浏览器无法打开 Google，提示"无法与服务器建立安全连接"。

---

## 🔍 排查步骤

### 步骤1：确认系统代理已开启

1. **在 Clash Verge 中**
2. **确认系统代理已开启**（按钮应该是开启状态）
3. **如果没开启，先开启系统代理**

### 步骤2：确认选择了正确的代理组

1. **在 Proxies 标签中**
2. **确认已选择 "Proxy" 代理组**（不是 "direct"）
3. **确认 "Proxy" 代理组中选择了 "HongKong-Squid"**

### 步骤3：检查代理是否运行

在终端运行：

```bash
curl -x http://47.243.177.166:3128 -I https://www.google.com
```

如果返回 200 或 302，说明代理可以访问 Google。

---

## ✅ 解决方案

### 方法1：重新启动 Clash Verge

1. **完全退出 Clash Verge**（右键点击菜单栏图标 → 退出）
2. **重新启动 Clash Verge**
3. **选择 `hongkong` 配置**
4. **选择 "Proxy" 代理组**
5. **开启系统代理**
6. **重新测试访问 Google**

### 方法2：检查系统代理设置

1. **系统设置 → 网络**
2. **选择当前网络连接**
3. **点击"高级"**
4. **查看"代理"标签**
5. **确认 HTTP 代理已配置**（应该是 127.0.0.1:7890）

### 方法3：清除浏览器缓存

1. **Safari → 设置 → 隐私**
2. **清除历史记录和网站数据**
3. **重新访问 Google**

### 方法4：尝试使用 HTTP（非 HTTPS）

1. **访问**：`http://www.google.com`（注意是 http）
2. **看是否能打开**（可能会重定向到 HTTPS）

- 测试代理连接
