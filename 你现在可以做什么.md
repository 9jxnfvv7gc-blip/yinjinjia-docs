### 4. 配置服务器地址

#### 修改配置文件
打开：`lib/config.dart`

修改：
```dart

// 或使用域名
```

## ❓ 如果遇到问题

### 界面显示异常
- 检查是否有编译错误
- 运行 `flutter clean && flutter pub get`
- 重新运行

### 连接服务器失败
- 检查 `lib/config.dart` 中的服务器地址
- 确保服务器正在运行
- 检查网络连接

### 上传功能不工作
- 确保后端服务已部署
- 检查API接口是否正确
- 查看错误信息
