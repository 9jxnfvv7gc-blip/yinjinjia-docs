### 方案2：清理下载缓存并重试

#### 在Android Studio中：

1. **Tools → SDK Manager**
2. **SDK Tools标签**
3. **取消勾选失败的组件**
4. **点击"Apply"**
5. **重新勾选并下载**

#### 或清理缓存：

```bash
# 清理Gradle缓存
rm -rf ~/.gradle/caches/

# 清理Android SDK下载缓存
rm -rf ~/Library/Android/sdk/.downloads/
```

然后重新在Android Studio中下载。
