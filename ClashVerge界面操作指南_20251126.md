# 🎯 Clash Verge 界面操作指南（2025-11-26）

## 📋 你看到的界面说明

你在 **proxies** 标签中看到了：
- **proxy groups**：代理组选择
- **选项**：rule, global, direct, script
- **global**：全局代理
- **direct**：直连（不使用代理）
- **delay check url**：延迟检查 URL

---

## ✅ 操作步骤

### 步骤1：选择代理组

1. **在 proxy groups 中**
2. **找到 "Proxy" 或类似名称的代理组**（这是我们配置的）
3. **如果看不到 "Proxy"，查看是否有 "HongKong-Squid" 或类似名称**
4. **选择这个代理组**

### 步骤2：不要选择 "direct"

- **"direct"** 表示直连，不使用代理
- **不要选择 "direct"**
- **选择我们配置的代理组**（"Proxy" 或 "HongKong-Squid"）

### 步骤3：找到系统代理开关

系统代理开关可能在：

#### 位置1：主窗口顶部

1. **查看 Clash Verge 主窗口的顶部**
2. **查找"系统代理"、"System Proxy"或类似按钮**
3. **点击开启**

#### 位置2：菜单栏图标

1. **点击菜单栏的 Clash Verge 图标**（右上角）
2. **查找"系统代理"、"System Proxy"或类似选项**
3. **点击开启**

#### 位置3：设置页面

1. **点击"设置"或"Settings"标签**
2. **查找"系统代理"、"System Proxy"或类似选项**
3. **开启它**

---

## 🔍 如果找不到系统代理开关

### 检查 Clash Verge 是否在运行

1. **查看菜单栏是否有 Clash Verge 图标**
2. **如果没有，Clash Verge 可能没有完全启动**
3. **重新启动 Clash Verge**

### 查看主窗口

1. **查看 Clash Verge 主窗口**
2. **应该有多个标签**：Proxies, Profiles, Settings 等
3. **切换到 Settings 标签**
4. **查找系统代理选项**

---

## 📝 配置检查

### 确认代理组正确

1. **在 Proxies 标签中**
2. **应该能看到 "Proxy" 代理组**
3. **里面应该有 "HongKong-Squid" 选项**
4. **选择 "Proxy" 代理组**

### 确认配置已加载

1. **在 Profiles 标签中**
2. **确认 `hongkong` 配置已选择**
3. **如果有"刷新"或"Reload"按钮，点击一下**

---
