## 4. "上传新内容"按钮的可见性

### ✅ 已修复：只有管理员才能看到

#### 修改内容
- ✅ 添加了管理员模式配置
- ✅ 只有管理员才能看到"上传新内容"按钮
- ✅ 普通用户看不到上传功能

#### 配置位置
在 `lib/config.dart` 中：

```dart
/// 是否启用管理员模式
static const bool enableAdminMode = true;

/// 当前用户是否为管理员
static bool get isAdmin {
  // TODO: 从登录状态获取
  return true; // 改为false可以隐藏上传按钮
}
```

#### 如何控制
- **显示上传按钮**：`isAdmin = true`
- **隐藏上传按钮**：`isAdmin = false`
