# 🔧 解决USB连接问题

## 📖 什么是adb devices？

### adb devices是什么：

**adb** = Android Debug Bridge（Android调试桥）
- 这是连接Android设备和电脑的工具
- `adb devices` 是检查设备是否连接的命令

### 运行adb devices后应该看到：

**如果设备已连接**：
```
List of devices attached
<设备ID>    device
```

**如果设备未连接**：
```
List of devices attached
（空白，什么都没有）
```

## 🔍 当前问题

### 你遇到的情况：
1. ✅ 找到了USB调试
2. ✅ 启用了USB调试
3. ❌ 但电脑上看不到设备（`adb devices`没有显示设备）

---

## 🚀 解决步骤

### 步骤1：确认USB调试已启用

在手机上：
1. **设置 → 开发者选项**
2. **确认"USB调试"已启用**（开关是打开的）
3. **可能还需要启用**：
   - "USB安装"（如果看到）
   - "USB调试（安全设置）"（如果看到）

---

### 步骤2：重新连接设备

1. **断开USB线**
2. **等待5秒**
3. **重新连接USB线**

---

### 步骤3：选择USB模式

当重新连接时，手机上会弹出提示：

**选择"传输文件"或"MTP"模式**
- ✅ "传输文件"
- ✅ "MTP"
- ❌ 不是"仅充电"
- ❌ 不是"传输照片"

---

### 步骤4：允许USB调试

1. **连接USB线后**
2. **手机上应该弹出"允许USB调试"对话框**
3. **如果没有弹出**：
   - 断开USB线
   - 重新连接
   - 或关闭/重新打开"USB调试"开关

4. **如果弹出对话框**：
   - 勾选"始终允许来自这台计算机"（可选）
   - 点击"确定"

---

### 步骤5：检查Mac权限

如果Mac提示权限：
1. **系统偏好设置 → 安全性与隐私**
2. **允许adb或Android相关权限**

---

### 步骤6：重新检查连接

在终端运行：

```bash
adb kill-server
adb start-server
adb devices
```

**应该看到设备**。

---

## 🔧 如果还是看不到设备

### 解决方法1：检查USB线

- **尝试不同的USB线**（有些USB线只能充电，不能传输数据）
- **尝试不同的USB端口**
- **确保USB线支持数据传输**

---

### 解决方法2：检查手机设置

1. **设置 → 开发者选项**
2. **关闭"USB调试"**
3. **等待5秒**
4. **重新打开"USB调试"**
5. **重新连接USB线**

---

### 解决方法3：重启adb服务

```bash
adb kill-server
adb start-server
adb devices
```

---

### 解决方法4：检查设备管理器（Mac）

在Mac上：
1. **系统信息 → USB**
2. **查看是否识别到Android设备**

---
